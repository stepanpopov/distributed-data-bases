### Запуск

- `git clone https://github.com/stepanpopov/distributed-data-bases.git`
- `cd distributed-data-bases`
- запуск Docker-контейнера командой `docker-compose up --build`
- запуск `sudo ./setup_subscriptions.sh` для настройки подписок репликации


После этого клиент будет доступен на http://localhost:5000/
